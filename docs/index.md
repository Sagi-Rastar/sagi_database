# 欢迎

这里是 SagiRastar 用于存储各类他所感兴趣的技术相关的文档库。

旨在妥善管理平日所积累的各类技术相关笔记、参考文档等需要反复查阅的内容。

??? note "MKDocs速查"
    - `mkdocs new [dir-name]` - 创建一个新项目
    - `mkdocs serve` - 本地调试web界面
    - `mkdocs build` - 构建项目
    - `mkdocs -h` - 帮助
